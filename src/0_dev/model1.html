El modelo intenta predecir mediante una Regresión Lineal el tiempo de descarga en segundos de ciertos paquetes definidos en Mb, megabytes.
Para esto:

  1. Los datos de entrenamiento (train data) y los datos de prueba (test data), se define como constantes en JavaScript:
  1. Para su uso con TensorFlow.js, se convierten estas contantes a tensores teniendo en consideración las dimensiones. Esto es, se definieron 2 constantes, cada una con 20 muestras (samples), en donde cada dato de entrenamiento en Mb le corresponde un dato en segundos:

  1. Se define un modelo de tipo secuencial, con sólo una capa y una neurona. Este modelo se compila utilizando:

    • Un optimizador tipo ‘sgd’, o simplemente “gradiente descendiente estocástico”.
    • Una función de pérdida de tipo ‘meanAbsoluteError’.

  2. Debido a que JavaScript es un lenguaje interpretado capaz de ejecutar sólo un hilo de ejecución a la vez, se define una función asíncrona para realizar el ajuste del modelo mediante model.fit()

  1. Ejecuta una evaluación del modelo entrenado mediante model.evaluate()
  1. Ejecuta una predicción mediante model.predict(). Para esto, define primero ciertos datos como constantes, y utilizando la capacidad de tratamiento de los datos como objetos de TensorFlow.js, convierte estas constantes a tensores, para finalmente imprimir la predicción en la consola.
