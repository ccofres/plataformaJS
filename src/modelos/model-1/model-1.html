Introducción a TensorFlow.js

El siguiente ejemplo es una introducción a la forma de uso básica de TensorFlow.js en el navegador. En él, se presenta el entrenamiento de una simple regresión lineal, además de ciertos métodos básicos como el uso de tensores, la definición de un modelo a traves de la Layers API y el ajuste mediante model.fit() .
  • Descripción del modelo
  • Particularidades del entrenamiento
  • Editor y ejecución del modelo






Descripción del modelo

El modelo intenta predecir mediante una Regresión Lineal el tiempo de descarga en segundos de ciertos paquetes definidos en MB, megabytes.

Para esto:

  1. Los datos de entrenamiento (train data) y los datos de prueba (test data), se define como constantes en JavaScript:
  1. Para su uso con TensorFlow.js, se convierten estas contantes a tensores teniendo en consideración las dimensiones. Esto es, se definieron 2 constantes, cada una con 20 muestras (samples), en donde cada dato de entrenamiento en Mb le corresponde un dato en segundos:

  1. Se define un modelo de tipo secuencial, con sólo una capa y una neurona. Este modelo se compila utilizando:

    • Un optimizador tipo ‘sgd’, o simplemente “gradiente descendiente estocástico”.
    • Una función de pérdida de tipo ‘meanAbsoluteError’.

  1. Debido a que JavaScript es un lenguaje interpretado capaz de ejecutar sólo un hilo de ejecución a la vez, se define una función asíncrona para realizar el ajuste del modelo mediante model.fit()

  1. Ejecuta una evaluación del modelo entrenado mediante model.evaluate()
  1. Ejecuta una predicción mediante model.predict(). Para esto, define primero ciertos datos como constantes, y utilizando la capacidad de tratamiento de los datos como objetos de TensorFlow.js, convierte estas constantes a tensores, para finalmente imprimir la predicción en la consola.






Particularidades del entrenamiento

  • Generalmente para visualizar el entrenamiento es necesario hacer uso de la consola del navegador.
    • Para Google Chrome, esta se encuentra en Más herramientas >> Herramientas para desarrolladores
    • Esta consola está reflejada en el Editor para poder visualizar este entrenamiento sin tener que recurrir a abrir la consola.
  • TensorFlow.js dispone de acuerdo a la Introducción a TensorFlow.js de varias API's intermedias o Backend's que ejecutan el cálculo y entrenamiento de tensores.

        const trainData = {
          sizeMB:  [0.080, 9.000, 0.001, 0.100, 8.000, 5.000, 0.100, 6.000, 0.050, 0.500,
                    0.002, 2.000, 0.005, 10.00, 0.010, 7.000, 6.000, 5.000, 1.000, 1.000],
          timeSec: [0.135, 0.739, 0.067, 0.126, 0.646, 0.435, 0.069, 0.497, 0.068, 0.116,
                    0.070, 0.289, 0.076, 0.744, 0.083, 0.560, 0.480, 0.399, 0.153, 0.149]
        };
        const testData = {
          sizeMB:  [5.000, 0.200, 0.001, 9.000, 0.002, 0.020, 0.008, 4.000, 0.001, 1.000,
                    0.005, 0.080, 0.800, 0.200, 0.050, 7.000, 0.005, 0.002, 8.000, 0.008],
          timeSec: [0.425, 0.098, 0.052, 0.686, 0.066, 0.078, 0.070, 0.375, 0.058, 0.136,
                    0.052, 0.063, 0.183, 0.087, 0.066, 0.558, 0.066, 0.068, 0.610, 0.057]
        };
        //console.log(testData.timeSec); //Muestra datos ingresados manualmente por consola
        //console.log(testData.sizeMB);
        //console.log(trainData.sizeMB);
        //console.log(trainData.timeSec);

        // Convirtiendo datos a tensores:
        const trainTensors = {
          sizeMB: tf.tensor2d(trainData.sizeMB, [20, 1]),
          timeSec: tf.tensor2d(trainData.timeSec, [20, 1])
        };
        const testTensors = {
          sizeMB: tf.tensor2d(testData.sizeMB, [20, 1]),
          timeSec: tf.tensor2d(testData.timeSec, [20, 1])
        }
        const model = tf.sequential();
        model.add(tf.layers.dense({inputShape: [1] , units: 1}));
        model.compile({optimizer: 'sgd', loss: 'meanAbsoluteError'});
        model.summary();

        (async function() {
          await model.fit(trainTensors.sizeMB,
                          trainTensors.timeSec,
                          {epochs: 500});
                          })();
        //Evalua el modelo, utilizando la funcion de perdida,
        //de esta forma, mientras mas cercana a 0, mejor
        model.evaluate(testTensors.sizeMB, testTensors.timeSec).print();

        const smallFileMB = 1;
        const bigFileMB = 100;
        const hugeFileMB = 10000;
        model.predict(tf.tensor2d([[smallFileMB],[bigFileMB],[hugeFileMB]])).print();
