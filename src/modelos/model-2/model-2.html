Modificando la introducción a TensorFlow.js

El siguiente ejemplo es una modificación del ejemplo de introducción a TensorFlow.js. En este ejemplo se preseta una modificación para visualizar el entrenamiento de una red neuronal a través de la correcta configuración de callbacks .
  • Descripción del modelo
  • Editor y ejecución del modelo






Descripción del modelo

El modelo intenta predecir mediante una Regresión Lineal el tiempo de descarga en segundos de ciertos paquetes definidos en MB, megabytes. Para esto entrena una neurona y configura callbacks para la visualización de su entrenamiento.

  1. Define una función asíncrona para ejecutar de forma asíncrona el ajuste del modelo mediante model.fit().
    • Como tercer argumento de model.fit() se definen callbacks a ejecutar a cada término de época. Estos callbacks se definen para mostrarse a través de la consola (mediante console.log()), y muestran el número de época y el valor de la función de pérdida al final de cada época.

  1. Se define un modelo de tipo secuencial, con sólo una capa y una neurona. Este modelo se compila utilizando:

    • Un optimizador tipo ‘sgd’, o simplemente “gradiente descendiente estocástico”.
    • Una función de pérdida de tipo ‘meanAbsoluteError’.

  1. Los datos de entrenamiento (train data) y los datos de prueba (test data), se define como constantes en JavaScript.

    Para su uso con TensorFlow.js, se convierten estas contantes a tensores teniendo en consideración las dimensiones. Esto es, se definieron 2 constantes, cada una con 20 muestras (samples), en donde cada dato de entrenamiento en Mb le corresponde un dato en segundos:

  1. Ejecuta una predicción mediante model.predict(). Para esto, espera la ejecución de la función asíncrona definida anteriormente al implementar en este paso el método then().

    Los datos definidos como constantes, son convertidos a tensores usando la capacidad de tratamiento de los datos como objetos de TensorFlow.js, que convierte estas constantes a tensores, para finalmente imprimir la predicción en la consola y a una ventana emergente.

async function doTraining(model){
      const history =
            await model.fit(xs, ys,
                  { epochs: 300,
                    callbacks:{
                        onEpochEnd: async(epoch, logs) =>{
                            console.log("Epoch:"
                                        + epoch
                                        + " Loss:"
                                        + logs.loss);
                        }
                    }
                  });
}
