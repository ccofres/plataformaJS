Forma de uso y trabajo en TensorFlow.js

La forma de trabajo define el marco de desarrollo y uso de este librería, y es por lo tanto, la forma básica y principal de utilización y trabajo en la plataforma.

De esta forma, el proceso descrito a continuación puede tomarse como la forma básica de trabajo para la implementación de modelos con TensorFlow.js .

Descripción del proceso

Se distinguen básicamente 3 formas de implementación:

  • Implementación de un modelo entrenado en Python.
  • Implementación de un modelo entrenado en JavaScript.
  • Implementación de un modelo pre entrenado de TensorFlow.js.

Modelo entrenado en Python

  1. Diseño y entrenamiento del modelo en lenguaje Python, utilizando la API TensorFlow.
  2. Guardado del modelo en un tipo compatible con convertidor de
    modelos
  3. Conversion del modelo a formato web compatible, utilizando convertidor de modelos.
  4. Implementación del modelo en TensorFlow.js.

Modelo entrenado en JavaScript

  1. Diseño y entrenamiento del modelo en lenguaje JavaScript, utilizando la API TensorFlow.js
  2. Guardado del modelo en formato web compatible.
  3. Implementación del modelo en TensorFlow.js

Modelo pre entrenado de TensorFlow.js

  1. Implementación/carga del modelo en TensorFlow.js
  2. Entrenamiento del modelo en TensorFlow.js.
  3. Guardado del modelo en formato web compatible.
  4. Implementación del modelo en TensorFlow.js
